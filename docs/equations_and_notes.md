# Equations and Notes

## Receptor occupancy
\[
\theta = \frac{[L]}{K_d + [L]}
\]

## Hill response
\[
R(L) = \frac{L^n}{K^n + L^n}
\]

## Exponential decay
\[
M(t) = M_0 e^{-kt}
\]

## Signaling with input pulse
\[
\frac{dS}{dt} = \alpha I(t) - \beta S
\]

## Signaling with negative feedback
\[
\frac{dS}{dt} = \alpha I(t) - \beta S - \gamma F S
\]

\[
\frac{dF}{dt} = \delta S - \epsilon F
\]
